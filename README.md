## What this project does

This project trains **SimpleRNN** and **LSTM** deep learning models on 10 years of Tesla stock data and compares their forecasting accuracy across three prediction horizons. The results are surfaced through an interactive Streamlit dashboard that displays all prices in **Indian Rupees (₹)** with a live FX rate control.

The dashboard looks and feels like a Bloomberg terminal — ticker bar at the top, hairline borders, monospaced numbers, no decorative fluff. The model loads once and is cached, so after the first run everything is near-instant.

## Results

| Model | Horizon | RMSE (₹) | MAE (₹) | R² |
|---|---|---|---|---|
| **SimpleRNN** | 1-day | ₹2,473 | ₹1,608 | **0.8495** |
| LSTM | 1-day | ₹2,739 | ₹1,893 | 0.8155 |
| SimpleRNN | 5-day | ₹3,585 | ₹2,389 | 0.6865 |
| **LSTM** | 5-day | **₹3,404** | ₹2,475 | **0.7174** |
| SimpleRNN | 10-day | ₹4,012 | ₹2,801 | 0.6120 |
| **LSTM** | 10-day | **₹4,000** | ₹2,823 | **0.6143** |

> SimpleRNN wins at 1-day. LSTM wins at 5-day and 10-day — its memory gates retain longer context that SimpleRNN's vanishing gradients can't.

## Quickstart

```bash
# 1. Clone
git clone https://github.com/Tanyagarg08/tesla-stock-prediction.git
cd tesla-stock-prediction

# 2. Install
pip install -r requirements.txt

# 3. Generate model files (run Section 8 of the notebook first)
jupyter notebook Tesla_TSLA_Submission.ipynb

# 4. Launch dashboard
streamlit run app.py
```

> **Note:** `tsla_scaler.pkl` and `tsla_lstm_best.keras` are generated by running **Section 8** of the notebook. Run that once before launching the app.

## How to run on Streamlit Cloud

1. Push this repo to GitHub (with `tsla_scaler.pkl` and `tsla_lstm_best.keras` committed)
2. Go to [share.streamlit.io](https://share.streamlit.io) → **New app**
3. Select your repo · branch `main` · file `app.py`
4. Click **Deploy** — done

## Approach

**Data** — 2,416 trading days of TSLA OHLCV data. Missing values handled with forward-fill (no look-ahead bias). Target variable: Adj Close.

**Preprocessing** — MinMaxScaler normalisation to [0,1]. Sliding window sequences with 60-day lookback. 80/20 chronological train-test split (no shuffling).

**Models** — Two-layer SimpleRNN and LSTM with Dropout(0.2) regularisation and EarlyStopping. Hyperparameters tuned with GridSearchCV over units ∈ {32, 64}, dropout ∈ {0.2, 0.3}, lr ∈ {0.001, 0.0005}.

**Evaluation** — RMSE, MAE, R², MAPE across 1-day, 5-day, and 10-day horizons.

---

## Limitations

- Uses price history only — no news sentiment, macro indicators, or options flow
- Trained on 2010–2020; Tesla's business has changed significantly since
- INR conversion uses a fixed FX rate (adjustable in the sidebar)
- Prediction accuracy degrades with horizon (R² 0.85 → 0.61)
